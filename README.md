# Projekt hasła

Demo aplikacji Flask do prezentacji danych z hashami.

## Uruchomienie

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Otwórz `http://127.0.0.1:5000`.

## Co robi demo

- pokazuje tabelę z rekordami,
- pozwala szukać po wszystkich polach,
- ma przycisk „Odszyfruj”, który uruchamia bezpieczną symulację analizy,
- wyświetla panel ze szczegółami wyniku.
